## Tentang Proyek

Hazz Farm adalah proyek pengembangan perusahaan agritech yang menggabungkan praktik pertanian modern dengan teknologi digital.

Repository ini berisi source code, dokumentasi, desain sistem, serta roadmap pengembangan website dan platform Hazz Farm.
